jCountdown - jQuery Countdown Plugin Demos

Locale Demo

Setup

Include a locale file with jCountdown

<script src="../src/jquery.jcountdown.js"></script>
<script src="../locale/jquery.jcountdown-de.js"></script>

Setup your HTML

//HTML

<p id="countdown"></p>

Then pass the date to the plugin: 

//JavaScript
				
$("#countdown").countdown({
	date: "January 28, 2013 00:00",
	leadingZero: false,
	yearsAndMonths: true,
	weeks: true
});
