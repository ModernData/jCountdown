jCountdown - jQuery Countdown Plugin Demos

Server Time Demo - Offset option

Another way to use a servertime is to use the offset option. Just case the offset to the plugin to match the server time when counting down.

Setup

// HTML
<p id="time"></p>

// JavaScript				
var dateTo = "January 21, 2013 12:00:00";

$("#time").countdown({
	date: dateTo,
	offset: -8
});
